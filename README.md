# AgileFitness Skills

**Interactive fitness and wellness coaching.**

Based on the book *Fitness and Wellness for Modern Human* by Manjeet Singh (Certified Fitness Trainer, Certified Nutrition Coach, Author & Speaker), these skills give you a personal coach that assesses your situation, answers your questions, and guides you through proven frameworks for physical fitness, nutrition, weight loss, longevity, and more.

> *"Improve 1% a day. Just 1% compounded, that's 3800% a year. That's ENORMOUS improvement."*

---

## What Are Skills?

Skills are interactive coaching sessions you run inside [Claude Code](https://claude.ai/code). Instead of reading a book cover-to-cover, you type a command like `/nutrition` and get a personalized conversation: the skill asks about your goals, calculates your numbers, and gives you a tailored action plan.

Each skill:
- Starts with a **quick assessment** of where you are right now
- Delivers **personalized frameworks, formulas, and tips** based on your answers
- Warns you about **common mistakes** to avoid
- Ends with a **clear action plan** you can implement immediately

## How to Use

### Prerequisites

You need [Claude Code](https://claude.ai/code) installed. Claude Code is available as:
- CLI in the terminal
- Desktop app (Mac/Windows)
- Web app (claude.ai/code)
- IDE extensions (VS Code, JetBrains)

### Option 1: Install as a Plugin (Recommended)

```bash
# Add the skills to Claude Code
claude plugin add coachmanjeet/agilefitness
```

Once installed, open Claude Code and type any skill command:

```
/fitness-wellness
```

### Option 2: Clone and Use Locally

```bash
# Clone the repository
git clone https://github.com/coachmanjeet/agilefitness.git

# Navigate to the directory
cd agilefitness
```

Then open Claude Code in this directory and use the skill commands.

### Option 3: Quick Start (No Installation)

You can also just tell Claude Code to use a specific skill by referencing it:

```
Use the fitness-wellness skill from https://github.com/coachmanjeet/agilefitness
```

## Recommended Path

**If you're new to fitness or starting fresh:**

```
/fitness-wellness  -->  /physical-fitness  -->  /nutrition  -->  /my-blueprint
```

1. `/fitness-wellness` — Understand the holistic framework, find your "why", build motivation
2. `/physical-fitness` — Design your workout program based on your body type and goals
3. `/nutrition` — Calculate your calories and macros, choose a diet approach, plan meals
4. `/my-blueprint` — Put it all together into your personal daily system

**If you have a specific goal:**

| Goal | Start Here |
|------|-----------|
| Lose weight / burn fat | `/weight-loss` |
| Build muscle | `/physical-fitness` then `/nutrition` |
| Fix my diet | `/nutrition` |
| Sleep better / live longer | `/longevity` |
| Get a complete system | `/my-blueprint` |
| Just get started | `/fitness-wellness` |

## What Each Skill Covers

### `/fitness-wellness` — The Foundation
- **3 drivers of results:** Luck, Strategy, Actions (you control 2 of 3)
- **4 pillars of health:** Body, Mind, Heart & Soul, Financial
- **Internal motivation system:** Purpose, Habits, Perseverance, After-feeling
- **Dopamine management:** Choose high-quality happiness sources
- **8 ways to reduce stress/cortisol**
- **Habit replacement framework:** 4 introspection questions
- **Time value framework:** Hourly rate calculation, 4-quadrant calendar

### `/physical-fitness` — Training & Workouts
- **Body type assessment:** Ectomorph, Mesomorph, Endomorph
- **Exercise selection:** Free weights vs bodyweight vs machines (decision matrix)
- **Training principles:** Time Under Tension, compound vs isolation, HIIT vs steady-state
- **Dynamic warmup protocol** (replace static stretching)
- **Recovery framework:** Good pain vs bad pain recognition
- **Strength benchmarks:** Squat 1.5x BW, Deadlift 2x BW, etc.
- **Specialized plans:** Women's cycle-based training, plus-size, busy professionals, runners

### `/nutrition` — Master Your Diet
- **BMR Calculator:** Harris-Benedict formula with step-by-step walkthrough
- **TDEE Calculator:** Activity-level multipliers
- **4 diet types:** Low Carb (25:35:40), Zone (40:30:30), Keto (5:35:60), Depletion
- **Macro math:** Protein/carb/fat calorie conversions
- **Staple food lists** for protein, carbs, fats, and fiber
- **Sugar quit roadmap:** 24 hours → 3-5 days → 2 weeks
- **Nutrition label reading:** 3-step process
- **Supplement guide:** Protein, creatine, BCAA, ZMA, multivitamins
- **Eating out strategies** and Indian diet adaptations

### `/weight-loss` — Lose Fat & Keep It Off
- **Fat loss math:** Calculate exactly how many pounds to lose and how many days it takes
- **Mini-cut protocol:** 750 calorie deficit, ~2 weeks, no cardio needed
- **5 intuitive deficit rules** (no calorie counting required)
- **Spot reduction truth** and what actually works
- **5 golden rules for maintaining abs**
- **7-step long-term maintenance framework**
- **Emotional eating management**
- **Foods that help vs foods that hurt**

### `/longevity` — Optimize Your Healthspan
- **Top 10 causes of death** and how to prevent each one
- **6 health KPIs** with exact target ranges
- **23 natural sleep strategies**
- **9 signs of overtraining/under-recovery**
- **Heat vs cold therapy** decision framework
- **Breathing techniques:** Diaphragmatic, long exhale, breath hold training
- **Anti-aging tips** from 7 longevity experts
- **12-step highly productive day** framework
- **Travel fitness** and jet lag recovery

### `/future-of-fitness` — Where Health Is Heading
- **Healthcare + fitness integration** vision
- **Wearable tech** evaluation guide
- **Emerging tech:** Genomics, stem cells, 3D printing in healthcare
- **10 meta trends** for the decade
- **Role of personal trainers** in the future
- **5 eco hacks** for sustainable fitness

### `/my-blueprint` — Build Your Personal System
- **The author's actual daily routine:** Diet (~1685 cal), workouts, morning routine
- **Weekly schedule template** with flexibility built in
- **Water protocol** and stress management stack
- **Key mantras and principles**
- **Fill-in blueprint template** to create your own personalized system
